# 📖 YU-GI-OH <a name="about-project"></a>

**YU-GI-OH**  is a cool website that tells you all about the awesome playing cards from the Yu-Gi-Oh! anime. It's a place where you can learn and discover more about these cards. If you're a big fan of Yu-Gi-Oh!, this website is perfect for you to explore and learn about your favorite cards!

> Key Features of YU-GI-OH Website


Includes the following features...

- **Show the number of cards categorized by type**
- **For every type, show the number of cards categorized by race**

## 💻 Getting Started <a name="getting-started"></a>

> A new developer can use this project as a guide to creating a collection cards app, or any other kind of collections, that fetches data from an API.

To get a local copy up and running, follow these steps.


### Prerequisites

In order to run this project you need:

```sh
  npm start
```

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/mohametalmeari/yugi-cards.git
```

### Usage

How to open your project in the browser? 
Double click on HTML file and the HTML file will be open in your browser.

### Run tests

To run tests, run the following command:

```sh
  npm test
```

### Deployment

You can deploy this project using:

```sh
  npm install gh-pages
  npm run deploy 
```
